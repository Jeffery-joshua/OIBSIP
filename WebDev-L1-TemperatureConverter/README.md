# 🌡 Temperature Converter

## 📌 Project Description

A responsive temperature converter that converts Celsius, Fahrenheit, and Kelvin.

## 🚀 Features

- Celsius ↔ Fahrenheit
- Celsius ↔ Kelvin
- Fahrenheit ↔ Kelvin
- Input Validation
- Reset Button
- Responsive UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
